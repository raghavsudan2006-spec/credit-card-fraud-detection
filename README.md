A machine learning project to detect fraudulent transactions using XGBoost, Isolation Forest, and SMOTE, with proper tuning and threshold optimization for imbalanced data.

A machine learning project to detect fraudulent transactions using XGBoost, where hyperparameter tuning is done with RandomizedSearchCV, an anomaly score is generated using Isolation Forest, and class imbalance is handled using SMOTE. The model is evaluated using Precision-Recall and ROC curves, and threshold tuning is applied to balance fraud detection and false positives effectively.

Tech Stack:
Python, Scikit-learn, XGBoost, Pandas, NumPy
